# ☕ Coffee Sales Analytics Dashboard

## Overview

The Coffee Sales Analytics Dashboard is an end-to-end Microsoft Excel project developed to analyze sales performance across products, customers, and time periods. The project transforms raw transactional data into an interactive dashboard that supports business decision-making through data visualization and KPI tracking.

---

# Project Objective

The objective of this project is to:

- Analyze coffee sales performance
- Identify top-performing products
- Monitor revenue trends
- Understand customer purchasing behavior
- Create an interactive dashboard for business reporting

---
